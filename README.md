# 5Y 平台自助上课助手

> 不愿再碰。愿世上再无刷课。

## Preview

![preview](./assets/preview.png)

## Features

- 自动抓取课程观看章节
- 自助分章节点播，颗粒度到小节
- 平台记录观看进度，安全有效
- 关闭页面后自动记录当前进度，下次打开后继续观看
- 支持最小化 & 调整窗体大小
- 无隐私泄露，不记录用户信息

## Usage

### Requirements

- 5Y 平台
- 浏览器
- Tampermonkey
